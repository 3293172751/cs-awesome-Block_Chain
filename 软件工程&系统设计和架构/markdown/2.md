## 软件工程面临问题

一般来说，在软件工程中，我们往往会面临三类问题：

+ 性能问题（The Performant System Problem）
+ 系统的嵌入式问题（the Embedded System Problem）
+ 复杂领域建模问题（the Complex Domain Problem）。
