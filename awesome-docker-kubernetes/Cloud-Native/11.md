## rsync远程同步工具

**rsync主要用于备份和镜像，具有速度快、避免相同内容和支持符号链接的优点**

> 相比较scp,rsync速度更快，只对差异文件做更新

**第一次同步等同于拷贝**

**基本语法**

```
rsync -rav atguigu@hadoop01:/opt/module atguigu@hadoop02:/opt/module
```

| -v   | 详细模式输出                           |
| ---- | -------------------------------------- |
| -z   | 压缩文件                               |
| -o   | 保留文件原始所有者身份                 |
| -g   | 保留文件原始所有组身份                 |
| -p   | 保留文件原始权限信息                   |
| -b   | 备份目标文件                           |
| -r   | 递归目录文件（传输目录内的子文件）     |
| -d   | 不递归目录文件（不传输目录内的子文件） |
| -P   | 显示进度信息                           |
| -q   | 精简输出模式                           |
| -h   | 显示帮助信息                           |
